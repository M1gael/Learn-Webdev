Audio Element

Audio clips are embedded within a web page using the audio element

audio src="url" attributes

Where url specifies the source and attrubutes define how the audio should be handled by the browser

Attributes of Audio and Video Elements

autoplay Starts playing media as soon as it is loaded
controls Displays the player controls in the web page
loop Automatically restarts media after it has finished playing
muted Specifies that the audio output should be muted
preload="auto/metadata/none" to preload : media , to load entire clip : metadata , or to not preload entire clip : none

Audio Formats
MP3 , MP3 , (.mp3) , audio/mpeg
AAC , AAC , (.aac .mp4 .m4a) , audio/mp4
OGG , Vorbis , (.ogg) , adio/ogg
WAV , PCM , (.wav) , audio/wav

Nest several source elements within a single audio element to provide several versions of the same media file

[audio]
[source src="url1" type ="mime-type"] (source is self closing)
[source src="url1" type ="mime-type"]
[/audio]

Note [url1] , [url2] represent the destination files and [mime-type] the type of audio format


Media Player Styling

Media player appearence is determined by the browser itself .
CSS can be applied for width , borders ,drop shadows as well as filters and transformations.

NOTE: If a browser has no Multimedia support

Add embed as a last option for dated browsers that dont support HTML5 multimedia
[embed src="lorem.mp3"
width="200" height="200"
controller="yes" autoplay="no" /]

Implementing Videos in HTML5

Videos are embedded into a web page using the video element , and attributes that control the behaviour and appearance of the video , url1 and url2 are the possible sources of the video
[video attributes]
[source src = "url1" type = "video/mp4"/]
[source src = "url2" type = "video/webm"/]

To play this video clip your browser needs to support HTML5

[/video]

Reset Style Sheet

Reset the entire style sheet in css using [ *{margin:0 ; padding: 0;} ] "*" representing "all"

Page Layout Designs

Fixed layout

Size of the page and elements are fixed , usually using pixels
[Width :200px ; height : 150px]

Fluid layout

The width of the page elements are set as a percent of the avalable screen width
[Width :100% ; height : 50%]

Elastic layout

images and text are always sized in proportion to each other in em units

Centre a Block Element

Horizontal centering

Block elements can be centered horizontally within their parent element by setting both the left and right margin to auto
[margin-left : auto] [margin-right-auto]

Vertical Centering

Centering an element verically can be acomplished by displaying the parent element as a table cell and setting the vertical-align to middle
[height :80px;] [display : table-cell;] [vertical-align : middle;]

Creating a Float Element

This propperty specifies wether an element should move to the left , right , or not at all
[float : position/value]

none - the element does not float (Displays just where it occurs in text).
left - the element floats to the left of its container.
right - the element floats to the right of the container.
initial - sets the property to its default value.
inherit - inherits this property from its parent element.

Clear a Floating Layout

Css Clear is for indicating whether HTML elements should move below the floating element that come before it OR it specifies what should happen with the element that is next to a floating element. [clear:position; (applying the propperty)].

left - the element is pushed below left floated elements.
right - the element is pushed below right floating elements.
both - the element is pushed below both left and right floated elements.
none - the element is not pushed below left or right floated elements.
inherit - the element inherits the clear value from its parent.

Prevent Container Collapse

Container collapse - elements with float property are removed fom the document flow so the sizes stay unknown for the parent element (as nothing is contained in it) so it is set to 0.

Method 1: Using Overflow Property--[overflow:auto].
Method 2: Making Parent Floated --[float:left].
Method 3: Specifying height Explicitly --[hegiht:30px].
Method 4: Using display and clear properties --[display:table; clear:both].

Grid Based Layouts

The CSS Gid Layout Module offers a grid-based layout system , with rows and colomns , making it easier to design web pages without having to use floats and positioning.

Two Types of Table Cells

Fixed grids - every column has a fixed position (columns widths and margins are specified in pixels).
Fluid grids - provides support for devices with different screen sizes (columns widths are in percentages).

Positioning Styles

Position property specifies the type of positioning method used for an element [position: static , absolute , fixed , sticky , initial , inherit ]

static - default value.
absolute - the element is positioned relative to its first positioned.
fixed - the element is postioned relative to the browser window.
relative - the element is positioned relative to its normal position.
sticky - the element is postioned based on the users scroll position.
inherit - inherits the property from its parent element.

Relative postioning

Relative places an element relative to its current position without changing the layout around it
[position : relative;] [left:40px] [top:10px]

Absolute postioning

Absolute places an element relative to its parent's position and changing the layout around it
[position : absolute;] [left:100px] [top:100px]

Overflow content

Overflow is what happens when there is too much content for a container
[overflow:hidden][overflow-x(or -y):scroll][overflow:auto]

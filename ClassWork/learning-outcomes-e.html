Web Tables

  • HTML table structure consists of multiple table rows.
  • Each table row contains one or more table cells.
  • An optional caption.
  • Each row group consists of an optional head section, an optional foot section, and a series of rows.

table html element used to create a table
tr -create table rows
th -create column headings
td -table data

Example:
table
tr
th Name th/
th Surname th/
tr/
tr
td Migael td/
td Smit td/
tr/
table/

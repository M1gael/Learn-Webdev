Web Tables

  • HTML table structure consists of multiple table rows.
  • Each table row contains one or more table cells.
  • An optional caption.
  • Each row group consists of an optional head section, an optional foot section, and a series of rows.

table html element used to create a table
tr -create table rows
th -create column headings
td -table data

Example:
table
tr
th Name th/
th Surname th/
tr/
tr
td Migael td/
td Smit td/
tr/
table/

Migael Smit

Row Groups

Seperate your table into 3 groups namely:
thead - Table head
tbody - Table body
tfoot - Table foot
<table>
<thead>
<tr>
<th colspan="2">Name and Surname</th>
</tr>
</thead>
<tbody>
<tr>
<td>Migael</td>
<td>Smit</td>
</tr>
</tbody>
</table>

Now thead , tbody and tfoot can be styled seprerately as groups .

Column Grops

The [colgroup] tag specifies a group of one or more columns in a table for formatting.
[span] tag specifies the number of columns the group should span
Remeber : You can use [colgroup] with span to do multiple columns
or simply [col id="col1"] [col id="col2"] [col id="col3"] to do one at a time and leave the ones empty that you wont be working with

Apply table layouts to other HTML elements using the CSS [display] property

  • Some display values in CSS that can create a table layout for an element and its children are:
    • display: table; makes an element behave like a <table> element.
    • display: table-row; makes an element behave like a <tr> element.
    • display: table-cell; makes an element behave like a <td> element.
    • display: table-column; makes an element behave like a <col> element.
    • display: table-column-group; makes an element behave like a <colgroup> element.
    • display: table-header-group; makes an element behave like a <thead> element.
    • display: table-footer-group; makes an element behave like a <tfoot> element.
    • display: table-row-group; makes an element behave like a <tbody> element.
    • display: table-caption; makes an element behave like a <caption> element.
